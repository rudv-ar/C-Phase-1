
I have forged out a makefile for just running the programs, then automatically deletes the binaries produced.
Usage : See the Makefile

**1. Swap two numbers**
Ask the user for two integers, then print them swapped.
`Enter a: 3` `Enter b: 7` → `a = 7, b = 3`

**2. Odd or Even**
Read an integer and print whether it's odd or even.

**3. Simple calculator**
Read two numbers and an operator (`+`, `-`, `*`, `/`). Print the result using `if/else`.

**4. Biggest of three**
Read three integers and print the largest one.

**5. Grade classifier**
Read a mark (0–100) and print the grade: A (≥90), B (≥80), C (≥70), D (≥60), F (below 60).

**6. Area calculator (with functions!)**
Write separate functions `circleArea(float r)` and `rectangleArea(float w, float h)`. Ask the user which shape they want, read the dimensions, and print the area.

I have provided the explanations as inline comments for these codes..
